# Labor Law Auto-Sync

Automatically fetches official government labor law pages and uses Claude API to extract structured data, then diffs against current `countries.js`.

## Setup

```bash
npm install @anthropic-ai/sdk
export ANTHROPIC_API_KEY=sk-ant-...
```

## Usage

```bash
# Run all Tier 1 countries (SG, HK, AE, PH, MY, GB, MT)
node scripts/labor-sync/extract.js

# Run one country
node scripts/labor-sync/extract.js sg
node scripts/labor-sync/extract.js hk

# Run a Tier 2 country
node scripts/labor-sync/extract.js tw
```

## Output

- `scripts/labor-sync/output/labor-YYYY-MM-DD.json` — full extraction
- `scripts/labor-sync/output/diff-YYYY-MM-DD.md` — diff report

## Workflow

```
Run script → Review diff report → Verify changes → Update countries.js → Update existing-data.js
```

## Automating with GitHub Actions

Add `.github/workflows/labor-sync.yml`:

```yaml
name: Labor Law Sync
on:
  schedule:
    - cron: '0 2 * * 1'  # Every Monday 2am UTC
  workflow_dispatch:

jobs:
  sync:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - uses: actions/setup-node@v4
        with: { node-version: '20' }
      - run: npm install @anthropic-ai/sdk
      - run: node scripts/labor-sync/extract.js
        env:
          ANTHROPIC_API_KEY: ${{ secrets.ANTHROPIC_API_KEY }}
      - name: Create PR if changes
        uses: peter-evans/create-pull-request@v6
        with:
          title: "data: labor law sync ${{ github.run_id }}"
          body: "Auto-generated by labor-sync script. Review diff report before merging."
          branch: labor-sync-auto
```
